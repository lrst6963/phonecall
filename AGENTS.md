# AI 开发助手行为准则
## 代码风格与规范

# 非必要禁止执行和使用任何命令！！!

### 1. 注释规范
- **语言**: 必须使用**中文**。
- **格式**: 保持简洁，禁止长句、英文长文或带序号的注释。
  - 正确: `// 监听主进程消息`
  - 错误: `// 1. listen to main process`, `// this function handles ipc...`

### 2. 聊天方式
- **原则**: 与用户保持中文对话，禁止使用英文或其他语言。
- **具体实现**: 所有与用户的交互（包括问题、指令、代码展示等）都必须用中文进行。
